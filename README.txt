Simple Twitter Feed WordPress Plugin, friendly with developers!

== Description ==

WP Simple Twitter Feed it's a plugin that allows users to create easy twitter feeds. The tweets are received through Twitter API, this allows us not to slow down the front-end.

The plugin it's user and developer friendly, the user can choose from two main styles and the developers can use their own HTML code.
